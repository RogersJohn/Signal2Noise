## What Changed

After these reports were generated, the following balance changes were implemented:
1. **Excitement multiplier**: 1.5× → 2.0× for exciting cards
2. **Bluff penalties**: Flat -2 → Escalating -2, -3, -4, -5
3. **Advertise deception penalty**: Added -1 audience for false advertising
4. **Post-advertise evidence**: Players place 1 card after seeing advertisements

These changes significantly affected game balance, making these archived reports no longer representative of current gameplay.
